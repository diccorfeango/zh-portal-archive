# zh-portal-archive

## 项目简介

zh-portal-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态 HTML 文件的集合存储与展示用途。

## 目录结构

```
zh-portal-archive/
├── pages/          # 存放归档的 HTML 页面
├── assets/         # 页面引用的静态资源（可选）
└── index.html      # 仓库入口页面（可选）
```

- **pages/**：该目录下以子目录或文件形式存放归档的 HTML 页面，每个页面保持独立完整。
- **assets/**：如需共享资源（如图片、CSS、JavaScript 文件），可放置于此目录。
- **index.html**：可选的入口页面，用于导航或索引。

## 页面归档说明

- 归档页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 每个页面可能包含完整的样式与脚本，不依赖外部服务。
- 页面内容仅代表归档时的状态，不保证持续更新或兼容性。

## 使用方式

1. 克隆或下载本仓库。
2. 在本地使用浏览器打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
